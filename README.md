# Slide Generator

Input an excel file and generate a slide deck.

## Usage

1. Create an xlsx file with this format.

|      | Edit cells in yellow   |
| ---- | ---- |
| Row 1     |      |
| Row 2     |      |

2. Copy your xlsx file to the slides directory.

2. 

```bash
npm run start
```

4. Open <Your Excel FileName>.pdf
5. Profit?!
